# AviRadiology

Radiologlar uchun yangi avlod sun'iy intellekt platformasi — korporativ MedTech / AI SaaS veb-sayti.

## Texnologiyalar

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **UI:** Shadcn UI + Framer Motion
- **Icons:** Lucide React

## O'rnatish

```bash
npm install
npm run dev
```

Loyiha [http://localhost:3000](http://localhost:3000) manzilida ishga tushadi.

## Sahifa strukturası

- `/` — Bosh sahifa (Hero, Muammo/Yechim, Afzalliklar, Workflow)
- `/demo` — Demo so'rash formasi
- `/platforma` — Platforma haqida
- `/ai-imkoniyatlari` — AI imkoniyatlari
- `/klinikalar` — Klinikalar uchun
- `/integratsiyalar` — PACS/DICOM/RIS integratsiyalari
- `/xavfsizlik` — Xavfsizlik va maxfiylik
- `/kompaniya` — Kompaniya haqida

## Dizayn

- **Ranglar:** Medical White (#F8FAFC), Deep Tech Blue (#0284C7), Slate (#0F172A)
- **Uslub:** Apple-style UI, Siemens Healthineers estetikasi, glassmorphism
